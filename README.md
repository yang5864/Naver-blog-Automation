# 네이버 블로그 서이추 Pro

네이버 블로그 서로이웃 자동 신청 프로그램 (macOS)

## 📋 주요 기능

- 🔍 키워드 검색으로 블로그 자동 수집
- 🤝 서로이웃 자동 신청
- ❤️ 공감 자동 클릭
- 💬 댓글 자동 작성
- 🎨 iOS 스타일의 세련된 GUI
- ⚡ 빠른 처리 속도

## 🚀 설치 방법

### 방법 1: 빌드된 앱 사용 (권장)

1. `dist/NaverNeighborPro.app` 파일을 다운로드
2. 앱을 Applications 폴더로 이동
3. 시스템 설정 > 개인정보 보호 및 보안에서 앱 실행 허용
4. 앱 실행

### 방법 2: 소스 코드에서 실행

```bash
# 저장소 클론
git clone [저장소 URL]
cd 서이추

# 가상환경 생성 및 활성화
python3 -m venv venv
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt

# 프로그램 실행
python NaverNeighborPro_GUI.py
```

## 🔨 빌드 방법

```bash
# 빌드 스크립트 실행 권한 부여
chmod +x build.sh

# 빌드 실행
./build.sh
```

빌드된 앱은 `dist/NaverNeighborPro.app`에 생성됩니다.

## 📖 사용 방법

### 1. 로그인

1. 프로그램 실행
2. "로그인" 버튼 클릭
3. 네이버 아이디와 비밀번호 입력
4. 로그인 완료 확인

### 2. 검색 키워드 설정

1. "검색 키워드" 입력란에 원하는 키워드 입력
2. 예: "상생페이백", "블로그", "리뷰" 등

### 3. 서이추 메시지 설정

1. "서이추 메시지" 입력란에 신청 시 보낼 메시지 입력
2. 예: "안녕하세요! 서로이웃 신청합니다 😊"

### 4. 작업 시작

1. "작업 시작" 버튼 클릭
2. 프로그램이 자동으로:
   - 키워드로 블로그 검색
   - 블로그 ID 수집
   - 서로이웃 신청
   - 공감 클릭
   - 댓글 작성

### 5. 진행 상황 확인

- 왼쪽 패널의 로그 창에서 실시간 진행 상황 확인
- 오른쪽 패널에서 Chrome 브라우저 화면 확인

## ⚙️ 시스템 요구사항

- macOS 10.13 이상
- Google Chrome 브라우저 (최신 버전 권장)
- 인터넷 연결

## ⚠️ 주의사항

1. **일일 한도**: 네이버는 하루에 신청할 수 있는 서로이웃 수에 제한이 있습니다.
2. **로그인 세션**: Chrome 브라우저를 통해 로그인하므로, 로그인 상태가 유지됩니다.
3. **사용 책임**: 이 프로그램은 개인적인 용도로만 사용하세요. 네이버 이용약관을 준수하세요.
4. **보안**: 로그인 정보는 로컬에만 저장되며, 외부로 전송되지 않습니다.

## 🐛 문제 해결

### 앱이 실행되지 않을 때

1. 시스템 설정 > 개인정보 보호 및 보안에서 "확인 없이 허용" 클릭
2. 또는 터미널에서 실행:
   ```bash
   xattr -cr dist/NaverNeighborPro.app
   ```

### Chrome 연결 실패

1. Chrome 브라우저가 설치되어 있는지 확인
2. Chrome을 완전히 종료한 후 다시 시도
3. 터미널에서 다음 명령어로 Chrome 프로세스 종료:
   ```bash
   pkill -f "Chrome.*remote-debugging-port=9222"
   ```

### 로그인이 안 될 때

1. Chrome 브라우저에서 직접 네이버에 로그인해보기
2. 프로그램을 재시작
3. 로그인 버튼을 다시 클릭
